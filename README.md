## Testing Locally

Assuming [Node.js/npm](https://nodejs.org/en/) are installed:
* `npm install`
* `npm run server`
* visit `localhost:3000`

## Compiling SCSS

Assuming Ruby is installed:
* `sudo gem install sass`
* `npm run sass`

## webpack

After `npm install` completes successfully, type `webpack` to compile scripts.
